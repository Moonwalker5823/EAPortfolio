# Eric Askew — Portfolio

Personal portfolio site for Eric Askew, Full Stack Software Engineer specializing in JavaScript, React, Vue.js, and Laravel.

## Live Sections

- **Intro** — Hero section with animated floating divs and social links
- **Services** — Overview of design, engineering, and UI/UX capabilities
- **Experience** — Professional work history
- **Hobbies** — Personal interests with an animated orbital circle component
- **Portfolio** — Project showcase (BeamQL, MultiPartForm React, TruckStar, Digiheart)
- **Testimonials** — Colleague and collaborator reviews (Swiper carousel)
- **Contact** — Email form with validation, powered by FormSubmit

## Tech Stack

- **Framework:** React 18 + Vite 4
- **Animations:** Framer Motion
- **Styling:** CSS Modules (custom, no CSS framework)
- **Form Submission:** FormSubmit.co
- **Carousel:** Swiper.js
- **Dark Mode:** React Context API

## Getting Started

```bash
npm install
npm run dev
```
